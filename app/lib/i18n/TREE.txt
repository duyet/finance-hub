Key Features
============
✓ Bilingual support (English/Vietnamese)
✓ Automatic locale detection (cookie → header → default)
✓ Vietnamese VND: 1.234.560 ₫ (dot separator, parentheses for negative)
✓ English USD: $1,234.56 (comma separator, minus for negative)
✓ Date formatting: VN (DD/MM/YYYY), EN (MM/DD/YYYY)
✓ 1,341 lines of translations (400+ keys)
✓ Type-safe with TypeScript
✓ SSR/CSR compatible
✓ Language switcher with persistence

Usage Quick Reference
=====================
Server-side:
  import { getLocaleFromRequest, formatCurrency } from '~/lib/i18n';
  const locale = getLocaleFromRequest(request);
  formatCurrency(1234560, 'VND', locale) // "1.234.560 ₫"

Client-side:
  import { useI18n } from '~/lib/i18n/client';
  const { t, formatCurrency } = useI18n();
  t('nav.dashboard') // "Tổng quan" (vi) or "Dashboard" (en)
